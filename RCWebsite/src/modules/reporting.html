Reporting Module

Create and manage dynamic Custom Reports

The RC Reporting Module lets you compose easy to read and fully customizable reports in an efficient way. In conjunction with RC Platform storing and managing reports gets a lot easier. But the key advantage lies in its automation and templating capabilities.

Different to common reporting mechanisms RC Reporting works with templates. You only have to create a template and store it in the RC Template Editor. The Reporting module then transforms these into full reports. Data is automatically assigend the templates variables.

Reports are useful in many ways. They can be used as a real-time dashboard, viewed statically or exported to be used in other typical office applications.

Templating

Create Reusuable Templates

The first step in creating a report is creating a template. In the screenshot is an example of a template created in the RC Template Editor.

With the editors easy-to-use interface you create a layout and then assign variables to the cells by typing or using the data browser. The reporting module takes care of the rest.

Reporting

Build real-time Dashboards or Reports

In the Reporting modules view you load a template and decide what data should be asssigend to it by selecting filters. The engine populates the templates variables automatically and repeats the template if necessary.

There is no restriction in what data can be used to populate the report. Naturally any data stored in RC Platform can be used but if necessary you can also manually insert data.

Reusing templates for e.g. different sessions isn't a problem. You could also use this a dashboard which updates in real-time.
