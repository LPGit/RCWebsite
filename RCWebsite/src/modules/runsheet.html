Runsheet Module

A New Workflow

Through our years of experience we have seen lots of ways on how to handle your crucial run specific data. The most common setup is engineers using various Excel sheets. Every team or engineer uses his own unique design and variables.

This approach always had several practical problems. It is hard to keep all your files organized, transfer the data or change the design. Especially sharing your runsheets was impossible resulting in only one user being able to see and edit it at the same time.

That's why we created a new runsheet solution that stores the data into a database, gives you a modern interface and solves all the problems cusotm excel sheeets have.

But to make it useful for any team, we made it templatable and customizable. Engineers can create their own design and variables with the template editor and share these with others. Everyone uses the design which fulfills his requirements best.

Full Flexibility
Through Templating

The template editor is an essential part of the Runsheet module. It allows you to create your own runsheet design through a simple interface. You create numeric textboxes and link them to any variables that exist on the RC Platform like Tyre Set, Tyre Type, Wing Angle, Camber or Tyre Mileage.

Features

Cross Module

Data that was collected by other modules can be used in the Runsheet as well. Vice versa RC Runsheets data can be used in other modules.

Solid Storage

Storing data to a solid database makes it manageable. Searching for specific runsheets, even if they are from past seasons, or taking the database online isn't a problem.

Multi User Friendly

No need to send anyone excel sheets. As the data is stored to the database, anybody can access it in real time provided he has been granted permission by the owner.

Large Teams

Especially for large teams where multiple people need to access the data. Storing the data on a local server that can be accessed from anywhere is a great workflow improvement.

Templates

You can always change the look of the runsheet while keeping the data in place. Two people can also view the same data in different layouts.

Look And Feel

Use a modern look and feel without all the flaws that come with spread sheets.
