Math Module

Develop Numerical Models in RC Platform

RC Platform provides a powerful module to develop new mathematical formulas or perform other numerical calculations on your data.

The module lets you leverage any data that was collected by RC Platform. To make this as easy as possible we developed a new syntax in which you write your formulas. Together with RC's data plotting featuers this gives you the ability to quickly test and evaluate new ideas.

And if your new ideas evolve into something bigger exporting the results for other applications or embedding them into RC reports is always an option.

Syntax

Easy to Read and Write Syntax

The fluid syntax was specifically developed for usage with data channels and numerical math. Formulas are written in a functional and declarative style. Channels are interpreted as standard Vector Algebra elements.

Especially the Dot-Operator improves the readability. Formulas can be thought and read from left to right instead of nesting function calls.

Another huge improvement is the Where-Operator. It lets you easily select specific time segments and restricts the calculations domain. (see picture)

Editor

Prototype and Test

The Formula Editor environment helps you manage all your formulas directly in RC Platform.

You can write formulas in the editor. Use the data browser to search for variables and assign them.

Test your code against any available datasets on the fly with the built in data plotter.

Exporting

Developing formulas in RC Platform makes it easy to add graphs to your reports by exporting them as image. But you can also export the numerical data and load it into another application.
