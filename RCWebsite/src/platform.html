RC Platform

Concept

In modern motor sports data efficiency is rapidly becoming the most decisive factor for success. The amount of data produced is growing day by day. Therefore engineers need a modern software solution to handle the large amount of data.

To solve the problem as a whole we built a fast, flexible and easy to use platform. It follows a simple but effective process: Import data, process data and evaluate data.

RC Platforms modular design allows the adaption to any enviroment. It works in small scale just as well as in large scale. You can just use a single module as well as many.

What it is

RC Platform is a server software to handle your data and take advantage fomr it. Normally it is deployed on a standard computer and takes place in your Local Area Network on race events. Back in the workshop or office you can take it online and access it from the web. RC Platform can be seen as a three part process.

First of all RC Platfrom helps you store and collect data. It doesn't really matter what data, any dat can be saved. A lot of data can be collected automatically when setup e.g. timing oder vehicle data. For other data like like setup or runsheet data we have specialized modules lie the Runsheet Moduel that enhance your workflow drastically. To keep up with changing enviroments we designed RC in a very modular design. So if something new comes up because of changing enviroments we can always adapt by adding new modules.

The collected data is stored in a fast and reliable database in open and unified data format. But it doesn't stop with collecting and storing data. The data is analysed when stored. So data from one source references data from another source if they are naturally linked. For example you got a lap time from the timing system, your engineer has a runsheet for that lap and there is a video from the tv stream for that particular lap. Then RC Platform automatically detects and links these three data sets.

Of course the platform was developed to make you better. That's where data evaluation comes in play. There are a lot of different modules to utilize the data in a new way. CornerTimes helps you analyze your driver more efficiently. Export modules help you output data for different purposes like reporting or Wintax. And if something new is needed, it can always be developed.
